Step 1: Setup VS Code + Copilot

1

Install GitHub Copilot & Copilot Chat

Make sure VS Code is installed, then install GitHub Copilot and Copilot Chat extensions.

2

Open MCP Configuration

Open the Command Palette in VS Code:

Ctrl+Shift+P (Windows / Linux)
Cmd+Shift+P (macOS)

Type: GitHub Copilot: Configure MCP Servers (or search for "MCP")

Click it – this will open the MCP servers configuration view.

3

Add MCP Server

In the MCP servers view:

  • Click "Add server" (or "+ New server")
  • Fill in the fields:
    • ID / Name: home-assistant
    • Type / Transport: stdio
    • Command: npx
    • Args: -y, @coolver/home-assistant-mcp@latest
    • Environment variables:

      Click the input field to select and copy the key, or use the Regenerate button to create a new one.

  • Save the server configuration
4

Allow Server in Copilot Chat

Open GitHub Copilot Chat in VS Code (side panel with the Copilot icon).

Start a new chat. In the tools / MCP section you should now see a server called home-assistant.

If Copilot asks whether it's allowed to use this tool/server, click Allow.

5

Test Connection

Ask Copilot Chat: "Use the home-assistant tools to list my Home Assistant entities"

Or try: "Ask the home-assistant MCP server to inspect my current automations"
